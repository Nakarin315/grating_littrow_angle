# Littrow Angle Calculator (Diffraction Grating)

A simple Python utility to calculate the **Littrow angle** for a diffraction grating based on its line density and wavelength.

## 📌 Overview

The **Littrow configuration** is commonly used in optics and spectroscopy, where the incident and diffracted beams follow the same path. This script computes the Littrow angle using the standard grating equation:

The Littrow angle is calculated using the grating equation:

    sin(θ) = (m * λ) / (2 * d)

where:

* θ = Littrow angle  
* m = diffraction order  
* λ = wavelength  
* d = grating spacing
